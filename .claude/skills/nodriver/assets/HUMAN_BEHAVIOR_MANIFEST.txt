USAGE SUMMARY
================================================================================

QUICK START
-----------
from human_behavior import HumanBehavior
behavior = HumanBehavior()
await behavior.read_page(page, "normal")

INSTALLATION
-----------
1. Copy human_behavior.py to your project
2. Import: from human_behavior import HumanBehavior
3. Use: behavior = HumanBehavior()

CONFIGURATION
-----------
config = BehaviorConfig(
    reading_speed=1.0,
    interaction_style=InteractionStyle.NORMAL
)
behavior = HumanBehavior(config)

COMMON PATTERNS
-----------
- await behavior.random_delay(0.5, 2.0) - General pause
- await behavior.reading_pause(1.0) - Decision pause
- await behavior.read_page(page) - Read naturally
- await behavior.scroll_naturally(page) - Scroll naturally
- await behavior.type_with_rhythm(elem, "text") - Type naturally

NOTES
================================================================================

ETHICAL USE
-----------
This module is designed for legitimate automation use cases:
- Testing your own applications
- Natural automation of tasks you do manually
- Monitoring publicly accessible content (with permission)
- Bot detection testing (with permission)

NOT intended for:
- Circumventing rate limits or access controls
- Impersonating humans to violate terms of service
- Unauthorized web scraping
- Fraudulent activity

BEST PRACTICES
-----------
1. Use appropriate delay distributions (don't use uniform random)
2. Match configuration to your use case
3. Combine behaviors naturally (hover -> click, read -> scroll)
4. Use context-appropriate pauses
5. Consider content complexity when setting delays
6. Review ethical guidelines before use

INTEGRATION
-----------
Works well with:
- safe_type.py (text input)
- smart_click.py (clicking)
- element_waiter.py (element waiting)
- network_monitor.py (network monitoring)
- quick_start.py (browser setup)
